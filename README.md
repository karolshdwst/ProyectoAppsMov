# ProyectoAppsMov
Repositorio para el proyecto de aplicaciones móviles

AhorraMas/
├── App.js                    # Punto de entrada principal
├── AppNavigator.js           # Sistema de navegación personalizado
├── package.json             # Dependencias y configuración
├── app.json                 # Configuración de Expo
├── assets/                  # Imágenes y recursos
└── screens/                 # Pantallas organizadas por módulos
    ├── Autenticacion/       # Login, registro, bienvenida
    ├── Panel-Principal/     # Dashboard principal
    ├── Transacciones/       # Gestión de transacciones
    ├── Presupuestos/        # Gestión de presupuestos
    ├── Estadisticas/        # Reportes y gráficos
    └── Compartidos/         # Componentes reutilizables


Tema visual:

Colores principales: Grises oscuros (#2a2a2a, #3a3a3a, #6b7280)
Acentos: Verde para ingresos, rojo para gastos, blanco para botones principales
Tipografía: Clara y moderna con diferentes pesos
Layout: Diseño en tarjetas con bordes redondeados

Gestión de Estado
Estado local con useState en cada componente
Props drilling para comunicación entre componentes
